# 🧠 Task Manager (Full-Stack App)

A simple full-stack task manager built with **Node.js, TypeScript, Express, and React (Vite)**.

---

## 🚀 Features

- Create tasks
- Toggle task (complete / incomplete)
- Delete task
- REST API
- Frontend connected to backend

---

## 🛠 Tech Stack

**Backend:**

- Node.js
- Express
- TypeScript

**Frontend:**

- React
- Vite

---

## 📦 Installation

### 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/task-manager.git
cd task-manager
```

---

### 2. Backend setup

```bash
cd backend
npm install
npx ts-node src/app.ts
```

---

### 3. Frontend setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🌐 API Endpoints

| Method | Endpoint   | Description   |
| ------ | ---------- | ------------- |
| GET    | /tasks     | Get all tasks |
| POST   | /tasks     | Create task   |
| PUT    | /tasks/:id | Toggle task   |
| DELETE | /tasks/:id | Delete task   |
